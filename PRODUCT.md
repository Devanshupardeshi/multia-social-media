# Product

## Register

product

## Users

Social media operators, creators, agency teams, and business owners who need to monitor Instagram Reels performance during active publishing windows.

## Product Purpose

The dashboard turns Instagram Graph API data into a focused live operations view for Reels: current views, reach, interactions, saves, shares, high-performing reels, and update velocity.

## Brand Personality

Professional, precise, energetic.

## Anti-references

Avoid noisy social-media-template styling, vanity-only creator dashboards, neon crypto dashboards, and layouts that hide operational details behind oversized marketing visuals.

## Design Principles

- Lead with the metric a user came to check.
- Keep API state and data freshness visible.
- Make every dense table row scannable in under two seconds.
- Favor trustworthy restraint over decorative analytics theater.
- Preserve useful demo data so the product can be evaluated before credentials are connected.

## Accessibility & Inclusion

Target readable contrast, keyboard-accessible controls, clear focus states, reduced-motion support, and layouts that remain usable on narrow screens.
